# Git vs GitHub

## Git
- A version control tool
- Runs locally on my computer
- Tracks file changes and history

## GitHub
- A cloud-based hosting service for Git repositories
- Used to store and share Git projects online
- Helps with collaboration

## Key Difference
Git is the tool.
GitHub is a platform that uses Git.

## Example
I can use Git without GitHub.
I cannot use GitHub effectively without Git.

## One-Line Summary
Git manages my code history, GitHub hosts it online.
