# git status

## What is git status?

git status shows the current state of the repository.

It tells me:
- Which files are modified
- Which files are staged
- Which files are untracked

---

## Why is it Important?

This is the most frequently used Git command.

It helps me:
- Avoid committing wrong files
- Understand what Git is tracking
- Stay aware of project changes

---

## Basic Syntax

```bash
git status
